== Description ==

Integrate BittAds into your blog with this plugin for Wordpress. It automatically embeds tags and enables you to make regular content into targeted ads.

BittAds is a solution for website owners and online ad agencies that need advanced, but easy to use advertisement management software. You can serve up to 25 million ads per month for free.

The BittAds for Wordpress Plugin is made available under the GPLV3. Please remind however, the BittAds service itself is not open-source.

== Installation ==

This section describes how to install the plugin and get it working.

1. Download and extract the latest version of the plugin
2. Copy bittads-wp.php to your Wordpress directory over FTP (/wp-content/plugins)
3. Activate the plugin in your Wordpress (http://(your home)/wp-admin/plugins.php)
4. Go to settings, click BittAds and follow the instructions.
